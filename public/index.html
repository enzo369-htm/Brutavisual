Sitio de muestra

Muestra para el estudio creativo IBV, tomando como referencia sitios de estudios internacionales y el material audiovisual publicado en el Instagram de IBV. El propósito es ofrecer una base navegable del futuro sitio: un punto de partida para proyectar arquitectura, tono visual y prioridades de contenido, y co-crear el producto final junto a quienes lideran el negocio.

Implementación: HTML5 semántico; CSS3 (Flexbox, Grid, unidades vh/clamp(), capas con z-index, hero position: fixed + bloque que “sube” al scroll); tipografía vía Google Fonts (Inter, Oswald); tratamiento de texto con filtros SVG (feGaussianBlur) y sombras; vídeo en hero y secciones (autoplay muted loop playsinline); sin framework de UI ni bundler. Deploy: sitio estático servido desde public/ en Vercel, integración GitHub → CI (push a main).
